## Descripción
Este proyecto es un sistema básico de gestión de restaurante desarrollado en Python utilizando Programación Orientada a Objetos (POO). El sistema permite gestionar productos y clientes mediante una estructura modular organizada en modelos y servicios.

## Estructura del proyecto
- modelos: contiene las clases Producto y Cliente
- servicios: contiene la clase Restaurante que gestiona la lógica del sistema
- main.py: punto de entrada del programa

## Reflexión
La modularización del software permite organizar mejor el código, facilitando su mantenimiento, lectura y escalabilidad. Separar responsabilidades entre clases mejora la calidad del desarrollo y evita la duplicación de código.
